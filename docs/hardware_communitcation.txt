HARDWARE COMMUNICATION: (Ports)
======================
	In order to talk to computer hardware, we need to read and write data from and to the computer hardware. In order to do this we use the
following assembly instructions
	in(x)
	out(x)

	where (x) can be 'b' 'w' or 'l', standing for byte, word and long respectively.
